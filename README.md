# VMHS Reunion — Next.js app

This project implements the reunion dashboard, students list, events, and a shared gallery backed by Cloudinary.

Environment variables (set these in Vercel or .env.local locally):

- CLOUDINARY_CLOUD_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET
- OWNER_PASSWORD

Deploy to Vercel and set the env vars in the project settings.
